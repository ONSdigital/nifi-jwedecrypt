# JWE decrypt processor for Nifi

A Nifi processor to decrypt and verify encrypted JWTs

## Installation

- Copy `nifi-jwedecrypt-nar/target/nifi-jwedecrypt-nar-1.0.nar` to the `lib` in your Nifi installation.
- Restart Nifi
